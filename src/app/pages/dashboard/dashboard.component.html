AdminPro es un ejemplo de App para administrar usuarios

Creado por Martín Bernal

  • Se puede editar el perfil de usuario.
  • Diferentes opciones de visualización.
  • Crear Médicos y Hospitales.
  • Asignar Médicos a los diferentes Hospitales.
  • Cambiar las imágenes.
  • Realizar búsquedas de Usuarios, Médicos Y Hospitales.

Creado con Angular 10, Node.js, MongoDB, Bootstrap y Fontawesome
